# Add Data to a Table

```sql
insert into ${Name} (${first_column_name}, ${next_column_name}) 
values ('first_val', 'next_val');

## Example

insert into Users (create_date, user_handle, last_name, first_name) 
values ('2018-06-06', 'a0eebc99-9c0b-4ef8-bb6d-6bb9bd380a11', 'clark', 'tyler');
```
