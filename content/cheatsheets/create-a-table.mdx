Replace `${Name}` with a Title Case name, and  `column_name` with a snake_case name. `type` corresponds to a data type.

<CodeWithExample>

```sql
create table ${TableName} (
  ${column_name} type,
  ${next_column_name} type
);
```

```sql
create table Users (
  create_date date,
  user_handle uuid,
  last_name text,
  first_name text
);
```

</CodeWithExample>
